# LineRider
It's a 2D Unity project for GAD2017 class.

## Instructions
Run or build the project and use mouse input to draw lines. </br>


## Acknowledgements
* Line renderer </br>
* More info can be found: https://docs.unity3d.com/Manual/class-LineRenderer.html

## Built With
* Unity 2018.4.2f1
